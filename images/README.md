# Carpeta de Imágenes

Coloca aquí las imágenes para la página de donaciones.

## Imágenes necesarias:

### Hero Section
- **hero-community.jpg** (800x600px recomendado)
  - Imagen principal del hero: comunidad estudiando, celebraciones, etc.

### Impact Section
- **ayuda-social.jpg** (400x300px recomendado)
  - Imágenes relacionadas con ayuda social a familias
  
- **educacion.jpg** (400x300px recomendado)
  - Imágenes de programas educativos, estudiantes, etc.
  
- **cultura.jpg** (400x300px recomendado)
  - Imágenes de eventos culturales, tradiciones, comunidad
  
- **emergencias.jpg** (400x300px recomendado)
  - Imágenes relacionadas con ayuda de emergencia

### Testimonials
- **testimonial-1.jpg** (200x200px recomendado, formato circular)
  - Foto de persona para testimonio
  
- **testimonial-2.jpg** (200x200px recomendado, formato circular)
  - Foto de persona para testimonio

## Notas:
- Formatos recomendados: JPG o PNG
- Las imágenes se mostrarán automáticamente cuando las coloques en esta carpeta
- Si una imagen no existe, se mostrará un placeholder con instrucciones
